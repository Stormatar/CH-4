  • Hex uses a hashtag followed by a string of numbers and or letters.
  • RGB uses three sets of one to three numbers separated by commas.
  • HSL uses a degree 0 to 360, saturation 0% to 100%, and lightness 0% to 100%

Preferred color code format:

  • Hexadecimal

HWB summary:

  • HWB stands for Hue, Whiteness and Blackness. it uses a hue of 0 to 360, whiteness 0% to 100%,and blackness 0% to 100%

CMYK summary:

  • CMYK is a combination of cyan, magenta, yellow, and key(black). Printers often use CMYK color values
